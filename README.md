# 🚀 Veltrex.Devs

**Student-built tech studio creating real-world products.**

We build AI tools, web apps, and campus systems focused on solving actual problems — not just showcasing code.

---

## 🔥 Projects

### 🧠 Onyx

AI-based interview training platform that improves communication, confidence, and speaking skills.

### 📄 OfficeDocs

LLM-powered system that reads PDFs and answers user questions instantly.

### 🏫 SideQuest

Campus platform for food ordering, printing services, and student utilities.

### 📅 Timetable Planner *(In Development)*

Smart scheduling tool designed to help students manage their time efficiently.

---

## ⚙️ Tech Stack

* **Frontend:** Next.js, TypeScript, Tailwind CSS
* **Backend:** Node.js, APIs
* **AI:** LLMs, LangChain, Python
* **Deployment:** Vercel

---

## 🛠️ Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the project.
